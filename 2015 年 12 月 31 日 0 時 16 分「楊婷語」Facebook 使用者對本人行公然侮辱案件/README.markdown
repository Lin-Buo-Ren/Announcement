公然侮辱人者，處拘役或三百元以下罰金。

### 中華民國刑法第 310 條		
意圖散布於眾，而指摘或傳述足以毀損他人名譽之事者，為誹謗罪，處一年以下有期徒刑、拘役或五百元以下罰金。  
散布文字、圖畫犯前項之罪者，處二年以下有期徒刑、拘役或一千元以下罰金。
